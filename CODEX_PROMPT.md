# Codex 專案優化提示

請協助優化這個家庭任務管理 App。

技術棧：

- Node.js
- Express
- Multer
- HTML / CSS / JavaScript
- `data.json` 作為本機資料儲存
- `public/uploads` 作為照片上傳資料夾

優化方向：

1. 修正亂碼與壞掉的 HTML / JavaScript 文字。
2. 讓家長與孩子兩種模式都能順暢使用。
3. 加強後端輸入驗證與資料格式保護。
4. 限制照片上傳格式與大小。
5. 改善手機版介面與操作體驗。
6. 保留本機 JSON 儲存，不改成資料庫，方便初學者理解與部署。
7. 讓 README 清楚說明安裝、啟動、同 Wi-Fi 使用方式。
