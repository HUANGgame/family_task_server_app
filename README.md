# 家庭任務管理 Server App

這是一個可以在家中區域網路使用的家庭任務管理工具。家長可以建立孩子、派送任務、指定是否需要完成照片；孩子可以用手機登入自己的任務頁，上傳照片並標記完成。

資料會存在本機的 `data.json`，上傳照片會存在 `public/uploads`。

## 安裝與啟動

```bash
npm install
npm start
```

啟動後在瀏覽器開啟：

```text
http://localhost:3000
```

如果要讓同一個 Wi-Fi 內的手機使用，請查詢電腦的 IPv4 位址，例如 `192.168.1.23`，手機開啟：

```text
http://192.168.1.23:3000
```

## 功能

- 家長名稱設定
- 新增與刪除孩子
- 派送任務與補充說明
- 任務可選擇是否需要照片
- 孩子模式可上傳照片、完成任務
- 任務完成提示
- 每 5 秒自動同步資料

## 專案結構

```text
server.js            Express API 與檔案上傳處理
public/index.html    前端介面
data.json            本機資料
public/uploads       上傳照片
```
